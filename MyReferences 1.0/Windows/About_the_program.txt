Программа "MyReferences" конвертирует элементы Вашего библиографического списка согласно стандартам,
принятым в англоязычной научной литературе. В качестве примера были использованы требования, предъявляемые
к разделу "References" в журнале  Quaestio Rossica. https://qr.urfu.ru/ojs/index.php/qr/about/submissions#guidelines
К сожалению, автору данной программы было сложно учесть все возможные варианты синтаксиса, используемые в библиографических ссылках.
Поэтому для получения корректных результатов автор рекомендует Вам придерживаться следующих правил:

1) Не разделяйте инициалы авторов и редакторов пробелом (Это важно для корректрой работы используемых в программе алгоритмов)
2) Не используйте внутри названий книг, статей и сборников знаки препинания
3) Если результат получается некорректным, попробуйте изменить/упростить синтаксис библиографической ссылки

Для пакетного преобразования библиографических ссылок:
1) Создайте в папке с программой текстовый файл "Ref.txt" и скопируйте в него элементы библиографического списка
   (При создании файла в операционной системе MS Windows в большинстве случаев не требуется указывать расширение файла ".txt". При сохранении фала в MS Windows используйте кодировку ANSI)
2) Каждый элемент библиографического списка в файле должен начинаться с новой строки (В начале строк допускается использование нумерации в формате "1)" или "1.")
3) Запустите пакетное преобразование, для этого введите "п" и нажмите клавишу "Enter"
4) Результат преобразования должен появиться в папке с программой в файле "Ref_Eng.txt"
